# Minimal Node.js Docker Image with pm2

This image is built using the [mhart/alpine-node](https://hub.docker.com/r/mhart/alpine-node/) image.

Versions:
- `latest`, `2.2.1` (node v7.2.1)
- `2.1.6` (node v7.1.2)
- `1.0.2` (node v5.9.0)
- `0.15.9` (node v5.4.0)

Example:
- `docker run -ti --rm lucor/node-pm2 --version`
